Bootstrap footer

Bootstrap footer is an additional navigation for the website. It can hold links, buttons, company info, copyrights, forms and many other elements.

You can set the color of the footer by adding one of classes from our color palette.

Just like any other components of MDBootstrap , Footers are responsive by default.

Basic Footer

Add or remove .font-small class to the <footer> element to change the size of the fonts inside the Footer.
